# Release Notes

## v0.1.0 (Initial Release)

### Features

- **31 Skills** covering all aspects of TanStack Start development
  - File-based routing with type safety
  - Server functions and data loading
  - Forms with validation
  - SSR, streaming, and hydration
  - TanStack ecosystem integrations

- **12 Interactive Commands** for guided workflows
  - Brainstorming and planning
  - TDD workflows
  - Quality checks

- **Auto-Detection** of TanStack Start project configuration
  - Version detection
  - Package manager
  - Vite configuration
  - TypeScript settings
  - TanStack integrations (Query, Form, Table)

### Supported Versions

- TanStack Start 1.x (RC)
- Vite 5.x, 6.x
- npm, yarn, pnpm, bun
